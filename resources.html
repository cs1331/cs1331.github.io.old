Java

  • Download the Java Development Kit (JDK) Standard Edition (Java SE 8)
  • Install the JDK
  • Install the JDK using our instructions.
  • Install Ant
  • Java Language Specification
  • Online Java 8 API docs (I recommend that you download the documentation to your computer for offline access.)
  • Online JavaFX 8 API
  • Java Code Conventions - Sun/Oracle's official code conventions for Java. They're not perfect, and parts are dated (like the advice on make files), but most professional Java teams use these conventions. We've summarized and honed these conventions in our own CS 1331 Style Guide.
